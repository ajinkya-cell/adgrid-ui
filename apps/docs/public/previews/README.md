# Gallery Preview Assets Folder

Place all your component preview assets in this folder.

## Naming Convention
*   **Static Screenshots**: `<slug>-static.png` or `<slug>-static.webp`
*   **Hover Animations**: `<slug>-hover.webp` or `<slug>-hover.gif`

*(Example: `void-button-static.png` and `void-button-hover.webp`)*
