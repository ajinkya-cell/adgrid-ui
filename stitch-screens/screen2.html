MagneticButton

Beta Component

A high-performance interactive trigger that attracts towards the user's cursor within a defined radius. Perfect for CTAs that demand attention.

Sandbox Active

Installation

Terminal
npx void-ui add magnetic-button

Usage

Component.tsx
1import { MagneticButton } from "@void/ui";
2 
3export default function App() {
4  return (
5    <MagneticButton
6      strength={0.5}
7      radius={200}
8    >
9      Launch Mission
10    </MagneticButton>
11  );
12}

Props API

Property Type Default Description
strength number 0.3 Pull force multiplier (0-1).
radius number 150 Detection radius in pixels.
damping number 0.15 Elasticity of the return snap.
variant 'solid' | 'outline' 'outline' Visual style preset.
